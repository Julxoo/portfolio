# Article Exemple

Ceci est un article exemple au format MDX (Markdown + JSX).

## Pourquoi MDX ?

MDX permet d'écrire du contenu en Markdown tout en intégrant des composants React. C'est parfait pour un blog technique !

## Fonctionnalités

Vous pouvez utiliser :

- **Gras** et *italique*
- Des listes
- Des liens [comme celui-ci](https://nextjs.org)
- Du code inline `const example = true`

### Blocs de code

interface BlogPost {
  title: string;

  tags: string[];
