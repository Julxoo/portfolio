CRM immobilier commercial sur-mesure développé pour ATC Immobilier (Aix-en-Provence) en remplacement de Salesforce. Conçu et développé en autonomie complète : architecture, frontend, backend, base de données, intégrations tierces, déploiement et maintenance.

## Contexte

ATC Immobilier gérait son activité commerciale via Salesforce, une solution générique et coûteuse qui ne répondait pas aux spécificités de l'immobilier commercial. L'équipe avait besoin d'un outil sur-mesure, adapté à leurs processus métier : gestion d'emplacements, matching offres/demandes, génération automatique de documents, enrichissement de données.

J'ai conçu et développé le CRM en autonomie complète, de l'architecture à la mise en production. L'outil est utilisé quotidiennement par 5 collaborateurs pour gérer plusieurs milliers de biens, comptes et contacts.

## Architecture

Monorepo frontend/backend avec architecture Feature-Sliced Design (FSD). Frontend Next.js 16 hébergé sur Vercel, backend Fastify 5 sur Bun avec Drizzle ORM. Base de données PostgreSQL avec Supabase (Auth + Storage). Jobs asynchrones via BullMQ + Redis. Architecture entièrement refactorée en V2 pour améliorer scalabilité et maintenabilité.

## Stack technique

**Frontend** : Next.js 16 + React 19, shadcn/ui (Radix + Tailwind CSS 4), Zustand + TanStack Query v5, React Hook Form + Zod 4, Mapbox GL + Mapbox Draw, TipTap, XY Flow, Recharts, nuqs

**Backend** : Fastify 5 sur Bun, Drizzle ORM (+ Drizzle Kit), Zod 4, Supabase (Auth + Storage), PostgreSQL, BullMQ + Redis, Pino (logs structurés), Sharp (images), Googleapis, ExcelJS, Archiver

**Tests et qualité** : Vitest, Biome (lint/format)

**APIs intégrées** : Pappers (SIREN/SIRET), Kaspr (enrichissement contacts), Mapbox (géocodage + recherche), MySendingBox (courrier postal), Gmail API, Google APIs (Slides, Drive, Calendar), Ubiflow (export annonces), OpenAI (personnalisation IA)

## Fonctionnalités clés

**Modules core** : gestion emplacements (CRUD multi-photos, géolocalisation Mapbox, calculs surface pondérés), comptes (enrichissement auto SIREN/SIRET via Pappers), contacts (enrichissement auto via Kaspr), enseignes (reconnaissance marques par distance de Levenshtein), offres/demandes avec matching intelligent, lots, transactions, prospects, calendrier/activités, notes, todos, notifications.

**Carte interactive** : Mapbox GL avec outils de dessin, visualisation des biens, clustering, filtrage géographique.

**Automatisations** : enrichissement automatique entreprises (Pappers) et contacts (Kaspr), normalisation adresses (Mapbox), génération de fiches commerciales (Google Slides), envoi d'emails avec templates personnalisables (Gmail API), envoi de courrier postal (MySendingBox), matching intelligent demandes/offres (multi-critères, multi-zones géographiques avec rayon configurable), export automatique vers Le Bon Coin (flux XML Ubiflow), personnalisation de textes par IA (OpenAI).

**Infrastructure** : journal d'audit complet traçant toutes les actions utilisateurs, recherche globale, monitoring production avec logs structurés et health checks, intégration Google Drive pour le stockage de documents.

## Résultats

Solution en production remplaçant Salesforce. Gestion de plusieurs milliers de biens, comptes et contacts. Automatisation complète des processus critiques : enrichissement données, génération documents, emails, courrier postal, export annonces. 5 utilisateurs quotidiens.
